One Image, Five Glitches

This page uses the same flower image five times. Each section tranforms the image in a different way using CSS, inspired by the ideas from the Glitch Studies Manifesto.

Manifesto idea: The noiseless channel is a myth. Imperfection reveals the medium.
CSS idea: Extreme contrast, grayscale, blur, and skew.
Connection: The flower image is no longer clean or transparent. The distortion becomes part of the message.

Manifesto Idea: Bends and breaks can become a metaphor for difference.
CSS idea: Rotation, scale, shadow, and displacement.
Connection: The image looks pushed out of its normal position, as of the page itself has shifted.

Manifesto idea: A glitch interrpts the ordinary form of an image.
CSS idea: Cropping, object-fit, object-position, and hidden overflow.
Connection: Only part of the flower image is visible, so the viewer has to imagine what is missing.
